# WorkiFy — Система управління змінами з ШІ

## 🎯 Elevator Pitch

**WorkiFy** — веб-застосунок для управління робочими змінами персоналу з вбудованим ШІ-асистентом на базі **Google Gemini 2.0 Flash**. Менеджери та працівники можуть переглядати розклад, подавати заявки та виявляти конфлікти. ШІ-агент дозволяє керувати системою через природну мову українською — без кліків по формах. Асистент також допомагає формулювати професійні тексти заяв на відпустку та лікарняний.

---

## 👥 Цільова аудиторія

- **Менеджери / адміністратори** малого та середнього бізнесу, які складають графіки змін вручну
- **Працівники**, яким потрібно переглядати особистий розклад і подавати заявки
- Сфери: роздрібна торгівля, кафе/ресторани, охорона, медицина — будь-де, де є змінна робота

---

## 🤖 ШІ-паттерн: Function Calling / Agent

Проєкт реалізує паттерн **Function Calling (Tool Use)** з агентною петлею через Google Gemini API:

1. Користувач пише запит природною мовою
2. Gemini 2.0 Flash отримує запит + список доступних інструментів
3. Модель самостійно вирішує, які функції викликати (і в якому порядку)
4. JavaScript виконує реальні системні функції (`detect_conflicts`, `add_shift`, `generate_request_text` тощо)
5. Результати повертаються до моделі, яка формує фінальну відповідь

### Доступні інструменти ШІ:

| Інструмент              | Опис                                                                                                                             |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `get_all_shifts`        | Отримання всіх змін (з фільтром по працівнику)                                                                                   |
| `detect_conflicts`      | Виявлення всіх накладань у розкладі                                                                                              |
| `get_workload_stats`    | Аналіз годин навантаження по працівниках                                                                                         |
| `add_shift`             | Додавання нової зміни (тільки менеджер)                                                                                          |
| `get_pending_requests`  | Перегляд активних заявок                                                                                                         |
| `approve_shift_request` | Схвалення заявки на зміну                                                                                                        |
| `reject_shift_request`  | Відхилення заявки на зміну                                                                                                       |
| `check_shift_conflict`  | Перевірка конфліктів для конкретного часового слоту                                                                              |
| `generate_request_text` | генерація професійно сформульованого тексту заяви на відпустку/лікарняний/вихідний у різних стилях (офіційний, дружній, стислий) |

## 🚀 Запуск проєкту

### Вимоги

- Будь-який сучасний браузер (Chrome, Firefox, Edge)
- **Google AI Studio API ключ** (безкоштовно для тестування)
- Доступ до інтернету

### Покрокова інструкція

#### 1. Отримання API ключа Google Gemini

1. Перейдіть на **https://aistudio.google.com/app/apikey**
2. Увійдіть через Google акаунт
3. Натисніть **"Create API key"** → **"Create API key in new project"**
4. Скопіюйте ключ (формат: `AIzaSy...`)

> 💡 **Безкоштовний ліміт**: Google AI Studio надає 15 запитів/хвилину та 1500 запитів/день безкоштовно.

#### 2. Локальний запуск проєкту

**Через Live Server (VS Code)**

```bash
# Встановіть розширення Live Server у VS Code
# Відкрийте папку проєкту
# Клікніть правою кнопкою на index.html → Open with Live Server
```

> ⚠️ Не відкривайте `index.html` напряму через `file://` — браузер може блокувати API запити.

#### 3. Налаштування в застосунку

1. Запустіть застосунок у браузері
2. Увійдіть через тестовий акаунт (див. таблицю нижче)
3. Перейдіть до **🤖 ШІ-Асистент** у бічній панелі
4. Введіть ваш API ключ (`AIzaSy...`) і натисніть **Підключити**
5. Готово! Можете писати запити українською мовою

> 🔒 Ключ зберігається лише у `localStorage` вашого браузера — не передається третім сторонам.

## 💬 Приклади запитів до ШІ-асистента

### Базові запити

```
Покажи всі зміни на цьому тижні
Перевір конфлікти в розкладі
Скільки годин відпрацювала Анна?
Є активні заявки?
```

### Дії менеджера (тільки для admin@workify.ua)

```
Додай зміну для Каріни на 2026-02-22 з 09:00 до 17:00
Схвали заявку номер 1
Відхили заявку номер 2
```

### ✨ Генерація текстів заяв (нова функція!)

```
Напиши мені заяву на відпустку з 10 по 20 лютого через сімейні обставини
Склади офіційну заяву на лікарняний з 15 по 18 лютого
Сформулюй коротку заяву на вихідний 25 лютого для весілля друга
Напиши дружню заяву на відпустку з 1 по 14 березня
```

**Підтримувані стилі:**

- 📝 **Офіційний** — класична ділова мова з шапкою та підписом
- 🤝 **Дружній** — більш неформальний, але ввічливий тон
- ⚡ **Стислий** — мінімум слів, максимум суті

## 🏗️ Архітектура ШІ-агента

```
Користувач
    │
    ▼
[chat input] ──► runAgentLoop()
                      │
                      ▼
              Google Gemini 2.0 Flash API
              (gemini-2.0-flash-exp)
                      │
         Є виклики функцій (functionCall)?
                   │         │
                  TAK       НІ
                   │         │
                   ▼         ▼
            executeToolCall()  Фінальна відповідь
                   │
         ┌─────────┴──────────┐
         │    Реальні JS      │
         │    функції:        │
         │  get_all_shifts    │
         │  detect_conflicts  │
         │  add_shift         │
         │  generate_request_ │
         │  text              │
         │  approve_request   │
         │  ...               │
         └────────────────────┘
```

### Переваги Gemini 2.0 Flash

- ✅ **Швидкість**: найшвидша модель у лінійці Gemini
- ✅ **Безкоштовний ліміт**: 15 req/min, 1500 req/day
- ✅ **Підтримка Function Calling** з грудня 2024
- ✅ **Мультимодальність**: текст + зображення (на майбутнє)
- ✅ **Довгий контекст**: до 1 млн токенів

## 📝 Залежності

Проєкт **не потребує встановлення npm-пакетів**. Усі залежності завантажуються через CDN:

- Google Fonts (шрифти)
- Google Gemini API (зовнішній запит з браузера)

---

## 🔒 Безпека

- API ключ зберігається лише в `localStorage` браузера
- Запити до Gemini API йдуть напряму з браузера (CORS дозволений)
- Немає backend серверу — всі дані тільки на клієнті
- Для production рекомендується проксі-сервер для приховування API ключа
